🔬 Planned Features in Detail
🧠
Custom Model Fine-Tuning
YOLOv8 PyTorch Roboflow
Planning
What
Fine-tune YOLOv8m on a cafe-specific dataset with varied lighting, angles, and crowd densities to achieve 90%+ mAP on real-world cafe scenes.
How
  1. Collect and annotate 2,000+ cafe images via Roboflow.
  2. Augment with HSV shifts, mosaic, and flips.
  3. Fine-tune YOLOv8m for 100 epochs on a GPU cloud instance.
  4. Export to ONNX / TorchScript for deployment.
📡
Multi-Camera Fusion
OpenCV RTSP Threading
In Design
What
Support multiple concurrent camera streams and merge detections into a single occupancy count, eliminating double-counting in overlapping zones.
How
  1. Spawn one detector thread per RTSP/webcam source.
  2. Define overlap zones via a JSON config with camera homographies.
  3. Use IoU-based de-duplication across overlapping bounding boxes.
  4. Aggregate per-zone counts into a global occupancy total.
📈
Predictive Analytics Engine
Prophet Pandas Chart.js
Planning
What
Forecast future occupancy levels by time-of-day and day-of-week using Facebook Prophet, displayed as confidence-band trend charts in the dashboard.
How
  1. Persist at least 4 weeks of hourly occupancy records to InfluxDB.
  2. Retrain a Prophet model nightly via a background cron job.
  3. Expose a /api/forecast?hours=24 endpoint with upper/lower bounds.
  4. Render forecast bands as a shaded overlay on the existing Chart.js trend graph.
🔔
Multi-Channel Alert System
Twilio SMTP Slack API
In Design
What
Notify cafe managers via SMS, email, and Slack the moment occupancy exceeds or falls below configurable thresholds.
How
  1. Define alert rules (threshold %, channels, cooldown period) in a settings UI.
  2. On threshold breach, push to a Redis queue to avoid blocking the detector thread.
  3. An async worker dispatches via Twilio (SMS), smtplib (email), and Slack Webhooks.
  4. Log every alert to the DB with acknowledgment tracking for the manager UI.
🗺️
Heatmap Visualisation
OpenCV NumPy Canvas API
Backlog
What
Overlay a colour-coded density heatmap on the live video feed showing which zones of the cafe are most frequently occupied over the past hour.
How
  1. Accumulate bounding box centre-points into a NumPy density array per frame.
  2. Apply a Gaussian blur and normalise to 0–255.
  3. Map through a Jet colour palette and alpha-blend onto the video frame.
  4. Expose a toggle in the dashboard to switch between raw and heatmap view.
☁️
Cloud-Native Deployment
Docker Kubernetes AWS / GCP
Backlog
What
Package the entire system as Docker containers orchestrated by Kubernetes, enabling auto-scaling, rolling updates, and zero-downtime deployments.
How
  1. Write a multi-stage Dockerfile — builder stage installs YOLO deps, runner stage strips dev tools.
  2. Define Kubernetes Deployment, Service, and HPA manifests.
  3. Set up a CI/CD pipeline (GitHub Actions) to build, test, and push images to ECR/Artifact Registry.
  4. Use NGINX ingress with TLS termination for secure public access.
