About the Project

Real-Time Cafe Occupancy
Monitoring System

An intelligent, AI-driven solution that keeps your cafe safe, efficient, and welcoming.

🎯

Objective

Detect and count people inside a cafe in real-time using deep learning, display live occupancy stats, and trigger alerts when capacity thresholds are exceeded — all from a single camera feed.

🧠

AI Technology

Powered by YOLOv8 (You Only Look Once), the state-of-the-art object detection model by Ultralytics. Trained on millions of images, it detects people with high accuracy in real-time.

⚡

Performance

Optimized for CPU-based inference with frame skipping, 320px inference resolution, and the nano model variant — delivering smooth real-time detection without requiring a GPU.

📊

Live Dashboard

A modern glassmorphism dashboard shows live video, KPI cards, an animated circular occupancy meter, and a real-time trend chart — all updating every 2 seconds automatically.

🛠️ Technology Stack
🐍 Python 3.10+ Core Language
🔥 Flask Web Framework
👁️ YOLOv8n Object Detection
📷 OpenCV Video Processing
📈 Chart.js Data Visualization
🎨 Glassmorphism UI Frontend Design
⚙️ How It Works
01
Video Capture
OpenCV reads frames from webcam, RTSP stream, or uploaded video file in a background thread.
02
AI Detection
YOLOv8n runs inference on every 2nd frame at 320px resolution, detecting only the "person" class with 45% confidence threshold.
03
Counting & Annotation
Detected bounding boxes are drawn on the frame with confidence scores. The count is compared against max capacity to determine status.
04
REST API
Flask serves the annotated video as JPEG snapshots and exposes JSON endpoints for occupancy data, history, and settings.
05
Live Dashboard
JavaScript polls the API every 2 seconds and renders animated KPI cards, circular meter, and trend chart in the browser.
